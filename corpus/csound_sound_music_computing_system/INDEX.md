## Description

This comprehensive textbook covers the Csound sound and music computing system, including its architecture, implementation, opcodes, programming techniques, and applications.

## Contents

This corpus contains 248 extracted chapters from the book, including:
- Front Matter (Authors, Copyright, Preface)
- Main Chapters (1-13+) covering Csound fundamentals, architecture, signal processing, and advanced topics
- Reference sections with opcode listings and technical details
- Bibliography with 137 references

## Structure

The content has been extracted and organized into markdown files:
- `chapter_00.md` - Front Matter
- `chapter_01.md` through `chapter_247.md` - Book chapters, sections, and references
- `index.json` - Metadata and chapter index

## Usage

This corpus is designed for RAG (Retrieval-Augmented Generation) applications to provide detailed information about Csound programming, audio synthesis, and computer music techniques.
